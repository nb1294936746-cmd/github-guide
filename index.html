❓ 常见问题

Q：网站打不开 / 显示 404 怎么办？

A：刚部署后可能需要等 2-5 分钟才能访问。如果持续 404，检查仓库中是否有 index.html 文件，以及 Settings → Pages 中是否正确配置了部署源。

Q：GitHub 需要付费吗？

A：不需要。GitHub 个人账号免费使用，GitHub Pages 托管静态网站也是免费的，不限流量。

Q：别人能修改我的网站吗？

A：不能。只有你自己登录 GitHub 后才能修改仓库中的文件。别人只能查看，不能编辑。

Q：可以绑定自己的域名吗？

A：可以。在仓库 Settings → Pages → Custom domain 中填写你的域名，再去域名服务商设置 DNS 解析指向 GitHub 即可。这是可选的高级操作。

Q：想删除这个网站怎么办？

A：进入仓库 → Settings → 滚动到最底部 "Danger Zone" → Delete this repository。删除后网站就下线了。

📖 GitHub 术语表

术语含义
Repository（仓库）存放项目文件的地方，类似一个文件夹
Commit（提交）保存一次修改记录，类似"存档"
Push（推送）把本地修改上传到 GitHub
Pull（拉取）把 GitHub 上的最新内容下载到本地
Branch（分支）独立的修改线路，main 是主分支
GitHub PagesGitHub 提供的免费静态网站托管服务
Actions自动化任务，用于自动部署网站等
Fork（复制）把别人的仓库复制一份到自己的账号下
Star（收藏）给喜欢的项目点赞/收藏
Issue（议题）用来提问、反馈 Bug 或讨论功能
